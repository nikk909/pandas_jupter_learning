# Pandas Learning Project

Practice notebooks for learning pandas, based on Keith Galli's tutorial:

[Complete Python Pandas Data Science Tutorial](https://www.youtube.com/watch?v=2uvysYbKdjM&t=333s)

## About

I manually typed all the code from the video to get familiar with pandas foundational functions.

## Notebooks

- `pandas_from_ Keith Galli1-5.ipynb` — Basics (parts 1–5)
- `6-Handling Null Values.ipynb`
- `7-AggregatingData.ipynb`
- `8-Advanced_Functions.ipynb`
- `9-NewFunctions.ipynb`

## Data

Sample datasets are in the `data/` and `warmup-data/` folders.

## For Others

If you're learning pandas, I recommend watching the video and typing the code yourself — it helps a lot.
